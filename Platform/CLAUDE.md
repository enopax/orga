# Platform Project

This directory contains the **Platform** repository - the full-featured production platform for infrastructure provisioning.

## Repository
- **Name**: platform
- **Type**: Next.js 15 + TypeScript Web Application
- **Status**:  Production Ready

## Key Features
- User authentication (NextAuth.js v5)
- Organisation, team, and project management
- Resource provisioning with templates
- File management and storage
- Role-based access control
- RESTful API for external integrations
- Real-time deployment progress tracking
- Comprehensive testing strategy (130+ tests)

## Development
```bash
cd platform
npm run docker:dev && npm run dev
```
