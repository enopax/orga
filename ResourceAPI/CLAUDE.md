# ResourceAPI Project

This directory contains the **ResourceAPI** repositories - the script-based provider framework with testing tools.

## Repositories
1. **resource-api** - Express REST API with Bash Script Interface
2. **resource-api-frontend** - Vue.js 3 Testing Tool

## Status
 Production Ready

## Key Features
- Multi-provider architecture
- Script-based resource operations
- JSON Schema-driven configuration
- Optional authentication with API keys
- Provider discovery endpoint
- Custom endpoint support

## Development
```bash
# Resource API
cd resource-api
pnpm dev

# Resource API Frontend
cd resource-api-frontend
pnpm dev
```
